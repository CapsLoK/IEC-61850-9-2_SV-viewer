# SV Scout - Sampled Values Analyzer

Кроссплатформенное приложение для анализа и визуализации потоков выборочных значений (Sampled Values) согласно стандарту IEC 61850-9-2LE.

## Основные возможности

### 1. Одновременное исследование множественных потоков
- Поддержка нескольких потоков SV одновременно
- Автоматическое обнаружение новых потоков
- Идентификация по MAC-адресу источника

### 2. Поддержка стандарта IEC 61850-9-2LE
- **80 выборок на цикл** (для 50 Гц: 4000 выборок/сек)
- **256 выборок на цикл** (для 50 Гц: 12800 выборок/сек)
- Распарсивание ASN.1 BER кодированных данных
- Поддержка форматов INT32, FLOAT32, FLOAT64

### 3. Вычисление векторных величин
- **RMS (среднеквадратическое значение)** - вычисляется по скользящему окну выборок
- **Фазовые углы** - вычисляются с помощью DFT на основной частоте
- Окно Ханнинга для уменьшения спектральных утечек

### 4. Визуализация
- **Диаграмма векторов (Phasor Diagram)** - круговая диаграмма с векторами токов и напряжений
- **Осциллограммы** - отображение мгновенных значений по каналам
- Цветовая кодировка фаз (A-красный, B-зеленый, C-синий)

### 5. Регистрация в формате COMTRADE
- Экспорт в формат IEEE C37.111 (COMTRADE)
- Генерация CFG (конфигурация) и DAT (данные) файлов
- Сохранение последних захваченных данных

### 6. Детальная информация
- Просмотр отдельных мгновенных значений
- **Переходы через ноль** - детектирование zero-crossings
- Информация о качестве данных
- Счетчик пакетов и потерянные пакеты
- Синхронизация выборок

## Использование

### Режим работы

1. **Demo Mode** - генерация синтетических данных для демонстрации
   - 3 фазных тока (Ia, Ib, Ic)
   - 3 фазных напряжения (Ua, Ub, Uc)
   - Нулевые ток и напряжение (In, Un)
   - Частота 50 Гц
   - 80 выборок на цикл

2. **Реальный захват** - работа с сетевым интерфейсом
   - Выберите сетевой интерфейс из списка
   - Укажите multicast IP (по умолчанию 01.0C.CD.04.00.01)
   - Нажмите "Start Capture"

### Вкладки интерфейса

#### Waveforms (Осциллограммы)
- Отображение мгновенных значений по каналам
- Цветовая индикация фаз
- Автоматическое масштабирование

#### Phasors (Векторы)
- Круговая диаграмма векторов
- Таблица с RMS значениями и фазовыми углами
- Частота и качество сигнала

#### Details (Детали)
- Подробная информация о потоке
- Мгновенные значения по каналам
- Индикация переходов через ноль
- Тип данных каждого канала

#### Recording (Запись)
- Статус записи
- Кнопка сохранения в COMTRADE
- Опция автозаписи

#### Log (Журнал)
- Журнал событий с временными метками
- Сообщения об ошибках
- Информация о захвате

## Структура приложения

### Классы

- `SVStream` - представление одного потока SV
- `SVPacketParser` - парсер пакетов IEC 61850-9-2LE
- `SVReceiver` - поток приема пакетов (QThread)
- `PhasorWidget` - виджет диаграммы векторов
- `WaveformWidget` - виджет осциллограмм
- `ComtradeWriter` - запись файлов COMTRADE
- `SVScoutWindow` - главное окно приложения

### Форматы данных

#### Пакет SV (упрощенно)
```
Ethernet Header (14 bytes)
  - Destination MAC: 01-0C-CD-04-xx-xx
  - Source MAC: 6 bytes
  - EtherType: 0x88BA
  
VLAN Tag (optional, 4 bytes)
  - TPID: 0x8100
  - TCI: VLAN ID + priority

ASDU (Application Service Data Unit)
  - svID: идентификатор потока
  - smpCnt: счетчик выборок (UINT16)
  - confRev: версия конфигурации (UINT32)
  - smpSynch: флаг синхронизации
  - DataSet: измеренные величины
```

#### COMTRADE файлы

**CFG файл** (текстовый):
```
Station name,Generated by SV Scout
1,8
1,Ia,A,R,1,1,1,1,1
2,Ib,A,R,2,1,1,1,1
...
01/01/2024,12:00:00.000
01/01/2024,12:00:01.000
F,50
L,L,100,1
32-bit IEEE floating point
```

**DAT файл** (бинарный):
- 32-bit IEEE floating point значения
- Последовательные отсчеты по каналам

## Алгоритмы

### Расчет RMS
```python
rms = sqrt(mean(samples^2))
```

### Расчет фазового угла (DFT)
```python
# Применяем окно Ханнинга
windowed = samples * hanning(N)

# DFT на основной частоте
real = sum(windowed * cos(2*pi*k*n/N))
imag = sum(windowed * sin(2*pi*k*n/N))

phase = -atan2(imag, real) [в градусах]
```

### Детектирование перехода через ноль
```python
if samples[-2] <= 0 < samples[-1]:
    zero_crossing_detected = True
```

## Настройки

### Samples/Cycle
- **80** - стандарт для 9-2LE (UCA Implementation Guide)
- **256** - повышенное разрешение (IEC 61869-9)

### Multicast IP
- Формат: 01.0C.CD.04.xx.xx
- Диапазон: 01.0C.CD.04.00.00 - 01.0C.CD.04.01.FF

## Примечания

1. Для работы в режиме реального захвата требуются права root/administrator
2. Приложение автоматически переключается в demo mode если:
   - Scapy не установлен
   - Нет доступа к сетевому интерфейсу
   - Не найдены пакеты SV

3. Поддерживаемые стандарты:
   - IEC 61850-9-2 (полный)
   - IEC 61850-9-2LE (UCA Implementation Guide)
   - IEC 61869-9 (современный стандарт)
