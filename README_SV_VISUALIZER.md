# IEC 61850 Sampled Values Visualizer

Приложение для визуализации потоков IEC 61850 Sampled Values (SV) с использованием библиотеки libiec61850 1.6.1.

## Возможности

- **Выбор сетевого интерфейса**: Автоматическое обнаружение доступных сетевых интерфейсов
- **Обнаружение SV потоков**: Мониторинг и обнаружение всех SV потоков на выбранном интерфейсе
- **Визуализация данных**: 
  - График значений данных в реальном времени
  - Фазорная диаграмма (Phasor Diagram)
- **Характеристики потока**: Отображение ключевых параметров SV потока
- **Детальная информация**: Полная информация о выбранном SV потоке

## Требования

- Python 3.8+
- PyQt5
- matplotlib
- libiec61850 1.6.1 (опционально, для лучшего качества разбора SV)

## Установка зависимостей

```bash
# Установка библиотек Python
pip install PyQt5 matplotlib numpy

# Компиляция libiec61850 с поддержкой Python (опционально)
cd libiec61850-1.6
mkdir build && cd build
cmake -DBUILD_PYTHON_BINDINGS=ON ..
make -j4
sudo make install
sudo ldconfig
```

## Запуск приложения

```bash
python3 sv_visualizer.py
```

**Примечание**: Для захвата SV пакетов могут потребоваться права root:

```bash
sudo python3 sv_visualizer.py
```

## Использование

1. **Выбор интерфейса**: В выпадающем списке выберите сетевой интерфейс, на котором ожидается наличие SV трафика
2. **Запуск захвата**: Нажмите кнопку "Start Capture"
3. **Выбор потока**: После обнаружения SV потоков выберите интересующий поток из списка
4. **Просмотр данных**:
   - Вкладка "Visualization": График данных и фазорная диаграмма
   - Вкладка "Characteristics": Краткие характеристики потока
   - Вкладка "Detailed Info": Подробная информация о потоке

## Структура приложения

- `sv_visualizer.py` - Основное приложение с графическим интерфейсом
- `sv_visualizer_backend.c` - C-библиотека для захвата SV (альтернативный бэкенд)

## Архитектура

Приложение использует многопоточную архитектуру:
- Основной поток: Графический интерфейс PyQt5
- Поток захвата: Обработка SV пакетов в фоновом режиме

### Поддерживаемые протоколы

- IEC 61850-9-2 Sampled Values
- IEC 61850-9-2LE (Light Edition)

## Формат данных

SV потоки содержат следующие параметры:
- **svId**: Идентификатор потока
- **datSet**: Ссылка на набор данных
- **appID**: Идентификатор приложения
- **smpCnt**: Счетчик сэмплов
- **confRev**: Ревизия конфигурации
- **smpMod**: Режим сэмплирования
- **smpRate**: Частота сэмплирования
- **refrTm**: Время привязки
- **dataValues**: Массив измеренных значений

## Troubleshooting

### Проблема: "pyiec61850 not available"
Решение: Убедитесь, что libiec61850 скомпилирована с опцией BUILD_PYTHON_BINDINGS=ON и установлена корректно.

### Проблема: "Permission denied" при захвате
Решение: Запустите приложение от имени root или настройте capabilities для Python:
```bash
sudo setcap cap_net_raw+ep /usr/local/bin/python3
```

### Проблема: Не обнаруживаются SV потоки
Решение:
1. Проверьте, что выбран правильный сетевой интерфейс
2. Убедитесь, что SV трафик действительно присутствует в сети
3. Проверьте настройки VLAN (SV часто используют VLAN tagging)
