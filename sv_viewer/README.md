# IEC 61850 Sampled Values Viewer

Приложение для просмотра и анализа IEC 61850 Sampled Values (SV) потоков с использованием библиотеки libiec61850-1.6.1.

## Возможности

1. **Сканирование сетевых интерфейсов** - автоматическое определение доступных сетевых интерфейсов
2. **Обнаружение SV трафика** - детектирование по типу кадра (EtherType 0x88BA) или диапазону MAC-адресов (01:0C:CD:04:xx:xx)
3. **Список SV потоков** - отображение всех обнаруженных потоков Sampled Values
4. **Визуализация данных**:
   - Векторная диаграмма (фазоры) для трёхфазных измерений
   - Осциллограмма значений во времени
5. **Детальная информация** - полная информация о выбранном SV потоке

## Зависимости

- libiec61850-1.6.1 (включена в репозиторий)
- GTK+ 3.0
- pkg-config
- libpcap (опционально, для низкоуровневого захвата пакетов)

## Сборка

```bash
# Сборка библиотеки libiec61850
cd libiec61850-1.6
make

# Сборка приложения sv_viewer
cd ../sv_viewer
make
```

## Запуск

```bash
# Требуется доступ к raw socket (root права или CAP_NET_RAW)
sudo ./sv_viewer
```

## Использование

1. **Выбор интерфейса**: В выпадающем списке выберите сетевой интерфейс, на котором ожидается SV трафик
2. **Запуск захвата**: Нажмите кнопку "Start Capture"
3. **Выбор потока**: В списке слева выберите интересующий SV поток
4. **Просмотр визуализации**:
   - **Overview & Visualization** - векторная диаграмма и осциллограмма
   - **Detailed Information** - детальная информация о потоке

## Технические детали

### Определение SV трафика

Приложение определяет SV трафик по следующим признакам:
- **EtherType**: 0x88BA (стандартный для IEC 61850-9-2)
- **MAC-адрес назначения**: Диапазон 01:0C:CD:04:00:00 - 01:0C:CD:04:01:FF (IEC 61850-9-2 LE)

### Поддерживаемые параметры SV

- SvID (Sampled Value ID)
- DataSet reference
- AppID
- Configuration Revision
- Sample Rate
- Sampling Mode
- Sample Count (SmpCnt)
- Reference Time (RefrTm)

### Формат данных

Приложение предполагает, что данные в ASDU закодированы как FLOAT32 (IEEE 754), что соответствует спецификации IEC 61850-9-2 LE.
